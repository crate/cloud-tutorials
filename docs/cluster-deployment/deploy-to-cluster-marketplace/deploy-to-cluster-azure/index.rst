=======================================
Deploy a cluster from scratch via Azure
=======================================

This tutorial explains how to get set up with CrateDB Cloud via our Microsoft
Azure cloud provider. It will walk you through the subscription via Azure
Marketplace, the configuration process via the CrateDB Cloud Console wizard,
and finally the deployment of your cluster.
