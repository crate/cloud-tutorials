=====================================
Deploy a cluster from scratch via AWS
=====================================

This tutorial explains how to get set up with CrateDB Cloud via our AWS cloud
provider. It will walk you through the subscription via AWS Marketplace, the
configuration process via the CrateDB Cloud Console wizard, and finally the
deployment of your cluster.
